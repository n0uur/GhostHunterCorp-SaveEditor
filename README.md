# GhostHunterCrop-SaveEditor
Ghost Hunter Crop Game Save Editor.

## How to use
1. Open save file.
2. Edit save file.
3. Save your file back to game directory.
4. Enjoy your easy life :smile:

__Backup your original `save.sav` file before doing anything__
